   Views  4.9K Development for Windows *Artificial Intelligence Old hardware IT-companies       Состоялся релиз стабильной версии клиента ChatGPT для Windows 3.1 с поддержкой стандарта шифрования TLS 1.3 для прямого подключения к серверу OpenAI. Проект получил названием WinGPT.Основной код приложения написан на языке программирования C с использованием стандартных возможностей Windows API. WinGPT распространяется под открытой лицензией GNU General Public License (GPL) v2 и работает с любой 16-битной или 32-битной версией Windows после Windows 3.1, включая Windows 3.11. Для активации доступа к ChatGPT необходим рабочий токен подключения к API OpenAI.Разработчик WinGPT пояснил, что в процессе создания клиента использовал стандартный набор элементов управления Windows 3.1. Единственный нештатный элемент пользовательского интерфейса — это строка состояния, которую разработчику пришлось реализовать вручную. Несмотря на наличие встроенной строки состояния в Windows 3.1, Microsoft не сделала её доступной для сторонних разработчиков. Поэтому создателю WinGPT пришлось использовать собственный код для достижения необходимого результата.«Память на машинах с Windows 3.1 весьма ограничена, поэтому я попытался уменьшить объём памяти, необходимый для WinGPT, особенно при отправке и получении запроса и ответа от API OpenAI. Ответы современных API в формате JSON не оптимизированы по размеру, и API OpenAI не является исключением. Я попросил модель быть краткой, чтобы ответы были как можно меньше. Я также решил не отправлять текст предыдущих чатов и вызовов API, хотя это означает, что бот не сможет использовать предыдущий контекст разговора», — рассказал автор WinGPT. В апреле разработчик под никнеймом FrankCYB опубликовал на GitHub код JavaGPT — клиента для запуска ChatGPT на Windows 98, ME, 2000, XP, Vista, 7 и более современных версиях Windows 8, 10 и 11. В марте разработчик и энтузиаст старых компьютерных технологий Йо Кхэн Мэн написал клиента ChatGPT для MS-DOS и запустил чат-бота из командной строки на своём портативном ПК на базе IBM 5155 с центральным процессором Intel 8088 с тактовой частотой 4,77 МГц и 640 КБ ОЗУ. Этот компьютер был выпущен в 1984 году, у него монохромный монитор, видеокарта CGA и откидывающаяся клавиатура. Клиент ChatGPT под MS-DOS работает через сетевое соединение (по шине ISA через совместимый адаптер NE2000) с современным ПК, подключённым к API OpenAI.
