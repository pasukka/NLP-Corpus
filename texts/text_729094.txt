               Amazon открыла бесплатный доступ к ИИ-помощнику CodeWhisperer — аналогу GitHub Copilot  Reading time  
    2 min
   Views  5.9K Programming *Amazon Web Services *Machine learning *Artificial Intelligence       Amazon сделала свой ИИ-помощник по кодированию CodeWhisperer бесплатным для всех разработчиков. Изначально он был доступен только для клиентов Amazon Web Services.В прошлом году Amazon запустила предварительную версию CodeWhisperer, которую разработчики могут использовать в различных интегрированных средах разработки (IDE), таких как Visual Studio Code, для создания строк кода на основе текстового запроса. CodeWhisperer автоматически отфильтровывает любые предложения кода, которые потенциально предвзяты, и помечает его. Он поставляется с функциями сканирования безопасности, которые могут выявлять уязвимости в коде разработчика, а также предлагать варианты их исправления. CodeWhisperer теперь поддерживает несколько языков, включая Python, Java, JavaScript, TypeScript и C#, Go, Rust, PHP, Ruby, Kotlin, C, C++, сценарии оболочки, SQL и Scala.Amazon также запускает продукт Bedrock, который поможет компаниям создавать и масштабировать приложения для генеративного ИИ. Он поставляется с множеством базовых моделей, на которые могут опираться разработчики, включая Claude от Anthropic, Stable Diffusion и Amazon Titan. Это должно облегчить создание инструментов на основе ИИ, которые генерируют текст, отвечают на вопросы, создают сводки и многое другое.GitHub, принадлежащий Microsoft, уже предлагает свой инструмент Copilot, который аналогичным образом генерирует код в среде IDE. Однако Microsoft сделала этот инструмент бесплатным только для студентов и разработчиков, работающих над популярными проектами с открытым исходным кодом. Остальные пользователи должны купить подписку за $10 в месяц. На днях GitHub представил инструмент Copilot для документов, который работает как ChatGPT и позволяет искать информацию об официальной документации библиотек, продуктов и ресурсов. Он охватывает ресурсы GitHub, React, MDN, Azure, TypeScript и Webpack.У DeepMind, компании Google, также есть собственный инструмент AlphaCode, но он пока находится в стадии тестирования. ИИ уже обучили понимать задачи программирования. Сама Google в 2022 году запустила секретный проект Pitchfork, в рамках которого она намерена обучить искусственный интеллект писать и исправлять код.
