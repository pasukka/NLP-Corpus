               Для системы 3D-моделирования Blender вышел плагин BlenderGPT  Reading time  
    1 min
   Views  6.5K Working with 3D-graphics *Machine learning *Artificial Intelligence       Разработчик под никнеймом gd3kr опубликовал на GitHub код плагина, добавляющего в систему 3D-моделирования Blender языковую модель GPT-4. Инструмент позволяет использовать естественный язык для управления процессом моделирования.После установки плагина в интерфейсе Blender появляется вкладка GPT-4 Assistant. В ней пользователи могут задавать команды на естественном языке. К примеру, языковую модель можно попросить создать объекты, поменять их размер и управлять другими параметрами.Принцип работы BlenderGPT строится на использовании официального API от OpenAI. Пользовательские команды передаются GPT-4, а в начале добавляется фраза «Напиши, пожалуйста, код Blender, который выполняет следующую задачу» (Can you please write Blender code for me that accomplishes the following task). После программа извлекает из ответа сгенерированный скрипт на Python и выполняет его в качестве сценария в Blender. Для работы с плагином необходимо получить API-ключ от OpenAI.Плагин написан на Python и распространяется по лицензии MIT. Проект открытый, и его код опубликован в репозитории GitHub.
