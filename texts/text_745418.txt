               GitLab представила собственного ИИ-помощника для разработчиков  Reading time  
    1 min
   Views  4K Programming *Machine learning *Artificial Intelligence       GitLab запустила открытый бета-тест Code Suggestions — ИИ-функции, помогающей разработчикам писать код. Code Suggestions поддерживает работу с 13 языками программирования.На сайте GitLab отмечается, что с помощью Code Suggestions пользователи могут генерировать фрагменты кода по текстовому описанию на естественном языке. К примеру, можно попросить написать шаблонный сниппет или тесты. Также в компании уделяют внимание безопасности и конфиденциальности. К примеру, команда GitLab утверждает, что не использует пользовательский код для обучения моделей.В основе Code Suggestions лежат модели с открытым исходным кодом, что позволяет компаниями лучше понимать принципы работы ассистента. Сейчас функция поддерживает работу с 13 языками: C/C++, C#, Go, Java, JavaScript, Python, PHP, Ruby, Rust, Scala, Kotlin, и TypeScript.Команда GitLab работает над расширением для Visual Studio Code и IDE на базе JetBrains IntelliJ, которое поможет получать доступ к Code Suggestions из редактора кода. Саму функцию анонсировали несколько месяцев назад.
