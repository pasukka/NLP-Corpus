Она использует технологии машинного обучения и искусственного интеллекта для генерации текста, отвечая на вопросы и общаясь с людьми на естественном языке.Возможности ChatGPT для бизнеса, в целом, безграничны. Он может быть использован для автоматизации клиентского обслуживания, обработки заказов, создания контента.OpenAI сделали удобный интерфейс общения с нейросетью, конечно до них тоже разрабатывали GPT модели, но именно они сделали удобный интерфейс общения с моделью в виде чата и дали возможность разработчикам создавать свои сервисы с помощью API.Зачем бизнесу ИИ и как его продавать?Бизнес всегда стремится к оптимизации. ИИ - это в каком-то смысле промышленная революция 21 века. Возможности модели подробно описаны на сайте OpenAI, осталось только определить проблему, которую сможет решить Chat GPT и как заменить уже рабочее решение проблемы более современным и удобным образом.Теперь находим проблему, с которой справится Chat GPT, например, написание текста для постов, создание сценариев для рилс, упаковываем ее в удобный интерфейс, например, в телеграм бот, к которому РФ давно привыкли и продаем бизнесу как альтернативу СММ специалистам или же инструмент для них по подписке.Проблема и решениеНа примере конкретного кейса продемонстрирую проблему и решение включающее использованием ИИ.Часто эксперты сталкиваются с проблемой написания контента для социальных сетей, ведь тренды меняются, отсюда вопросы: "как придумать оригинальный сценарий, текст для поста, идею для контента?".Мы разработали решение - бот в телеграме, который будет писать тексты для постов и сценарии для рилз за учеников. Это может помочь им определиться со структурой и стилистикой поста. Также мы предложили ввести этот продукт по подписке для учеников.Также, как сайд-эффект мы стремились увеличить охваты, за счет того, что мы разрабатывали продукт под самого блогера, со своими нюансами обработки запроса пользователя и форматом вывода ответа, а не просто голый ChatGPT в ТГ.Наша цель была понять как в реальном проекте может быть полезен ИИ как часть системы.Что под капотом и реализация продуктаИтак, мы определились, что нужно создать Web App в Telegram, который будет писать сценарии для Reelz и тексты для постов, используя Chat GPT API для обработки запроса пользователя и ответа в определенном формате.У Chat GPT и у Телеграм ботов есть API, есть библиотеки для популярных языков, можно пользоваться http запросами - простор для творчества одним словом.СтэкLaravel для бэкэнда, разработка API для связи бота с Chat GPTWeb App будем писать на Vue.jsСервис контроля подписок и платежная системаLaravel Jobs для очередейMySQL храним информацию о подписке пользователя и др.Redis для отправки ответа после успешно выполненной джобыПланОпределиться с идеей проекта, который будет возможно быстро реализовать и он будет использовать ИИ для решения проблемы бизнеса.Создать удобный интерфейс взаимодействия с ИИ. Сделать понятный интерфейс статуса подписки, кнопки подписки, отписки и поддержки.РазработкаWeb App в Телеграм (тут есть подробная статья про это)Чтобы любой сайт стал web app, достаточно просто подключить скрипт.<script src="<https://telegram.org/js/telegram-web-app.js>"></script>После этого нам будет доступен объект: window.Telegram.WebApp. Так выглядит интерфейс в Telegram, так же он выглядит и в обычном браузере, за исключением доступности объекта  window.Telegram.WebApp.Именно этим объектом мы пользуемся, когда получаем информацию о пользователя у себя на сервере.Вот пример функции получения пользователя с сервераasync getProfile(){    const data = {        tg_id: window.Telegram.WebApp.initDataUnsafe?.user?.id,    };        await axios.get("API_BASE", {params: data}).then((response) => {        this.profile = response.data;    });};При открытии Web App мы сразу обращаемся к серверу и ищем пользователя с id который есть у нас в объекте window.Telegram.WebApp, если пользователь есть в нашей базе данных, возвращаем в Web App, если нет, то создаем и возвращаем его.Авторизация прошла успешно, пользователь даже не заметил.Мы храним еще и историю запросов, чтобы пользователь мог отредактировать его или запустить заново, если сервер вернет ошибкуДелаем бэкэнд для связи Web App и Chat GPTОпущу все технические моменты общения Chat GPT и Web App через сервер, но логика такая:Web App отправляет на сервер запрос о генерации поста на тему котовСервер запускает джобу на отправку запроса к Chat GPTНо Chat GPT плохо работает с русским языком.Нам нужно сделать так, чтобы Chat GPT обрабатывал запросы на английском.Не проблема, подключаем Яндекс Переводчик.Получаем промпт пользователя, переводим его на английский, отправляем запрос на английском в Chat GPTПолучаем ответ на английском и снова переводим ответ и отправляем его обратно в WebApp.Пример ответа от Chat GPT, который сервер доставил прямо в ботаСупер. Это особенно полезно, если у вас ограниченные ресурсы для контент-маркетинга.Комментарий: Определенно - да, но само собой текст требует небольшой модерации.Пример:Запрос: Напиши трехдневный контент-план для Instagram блогера, который пишет о нейронных сетяхОтвет:Конечно, я могу помочь вам с этим!
