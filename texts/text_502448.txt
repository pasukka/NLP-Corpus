Офис IBM продолжает серию еженедельных семинаров от ведущих российских и европейских экспертов. На это неделе нас ждем много всего интересного:


18 мая 10:00-18:00 Переход к микросервисам: возможности DevOps и инструментария модернизации приложений в IBM Cloud Pak for Applications [ENG]


Описание
С IBM Cloud Pak for Applications вы легко сможете использовать кластер Red Hat OpenShift для разработки, модернизации и развертывания новых Cloud Native приложений и микросервисов.

* Онлайн — семинар будет проводиться на английском языке! 


19 мая 11:00-13:00 Практический мастер-класс по Data Science в облаке IBM — от Эммануэля Женар [ENG]


Описание
 Разработка, обучение и развертывание моделей ИИ с примерами на базе IBM Watson Studio. Мастер-класс по использованию сервисов в облаке IBM: Watson Studio, Watson Machine Learning, AutoAI.

* Онлайн — семинар будет проводиться на английском языке! 


19 мая с 15:00-15:30 Сервисы IBM для использования открытого программного обеспечения в архитектуре предприятия


Описание
Развитие инфраструктуры на базе ПО с открытым исходным кодом затрагивает и вовлекает в процесс различные функции компании — помимо DevOps инженеров, архитекторов и руководителей проектов зачастую необходимо участие юристов, сотрудников безопасности и руководителей бизнес-подразделений. Одной из наиболее важных задач в таком случае является организация всех этих бизнес-функций с разными точками зрения в набор согласованных процессов. На вебинаре обсудим корпоративную стратегию и управление ПО с открытым исходным кодом, разберем примеры, а также поговорим о том, как IBM может помочь на каждом этапе работы с открытым ПО. 


20 мая 10:00-18:00 Сервис Watson Discovery: работаем с неструктурированными данными [ENG]


Описание
 IBM Watson Discovery — технология поиска на основе ИИ, которая извлекает информацию из неструктурированных данных. Используя последние достижения в области машинного обучения и обработки естественного языка, Watson Discovery упрощает загрузку и анализ данных для компаний без необходимости глубоких знаний Data Science. Watson Discovery легко обучается для конкретных областей бизнеса, а благодаря гибкости IBM Cloud разворачивается за минуты по требованию.

* Онлайн — семинар будет проводиться на английском языке!

IBM Cloud Pack for Integration предлагает упрощенное решение задач интеграции, позволяет быстро модернизировать свои бизнес-процессы для будущих инноваций. Интересно?
