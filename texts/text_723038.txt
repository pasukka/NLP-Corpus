               Microsoft анонсировала Microsoft 365 Copilot — офисного чат-бота на базе GPT-4  Reading time  
    1 min
   Views  4K Machine learning *Artificial Intelligence IT-companies       Microsoft анонсировала Copilot для набора офисных приложений Microsoft 365. Чат-бот поможет пользователям генерировать тексты, письма, перефразировать отрывки, составлять графики и анализировать данные в таблицах. Пока Copilot для офиса тестируется ограниченным количеством пользователей.Microsoft 365 Copilot построен на базе языковой модели GPT-4 и будет входить во все приложения офисного пакета Microsoft, включая Word, Excel, Power Point, Outlook и Teams. В компании рассказали, что Copilot поможет упростить рутинные задачи пользователей. Чат-бота можно будет попросить составить черновик по заданной теме, выделить главное из документа или перефразировать отрывок.В Power Point будет возможность генерировать презентации. В демонстрационном ролике показали, как Copilot создаёт презентацию, используя в качестве основы другой документ с информацией. Пользователи Ecxel смогут быстрее анализировать информацию, искать закономерности и визуализировать данные.На этой неделе похожие ИИ-инструменты для пакета офисных приложений представила Google. Пользователи также смогут использовать возможности языковой модели для ускорения рабочего процесса и автоматизации рутинных задач.
