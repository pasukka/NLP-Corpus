               OpenAI выложила на GitHub проект OpenCopilot  Reading time  
    1 min
   Views  17K Open source *SaaS / S+S *GitHub *Artificial Intelligence       Компания OpenAI опубликовала на GitHub проект OpenCopilot в ранней бета-версии. Пока он доступен для ограниченного использования.OpenCopilot позиционируется как ИИ-помощник, который можно встроить в свой продукт. Он интегрируется с базовыми API и может выполнять вызовы API, когда это необходимо. OpenCopilot использует большую языковую модель, чтобы определить, требует ли запрос пользователя вызова конечной точки API. Затем он решает, какую конечную точку вызывать, и передает соответствующую полезную нагрузку на основе данного определения API.В настоящее время OpenCopilot поддерживает Swagger OpenAPI 3.0. В будущем разработчики предложат удобный пользовательский интерфейс.OpenAI анонсирует, что OpenCopilot позволит пользователям платформ взаимодействовать с ними с помощью текстовых подсказок, а также обеспечит их круглосуточную поддержку. Вскоре появится возможность настраивать сложные сценарии взаимодействия. Со временем в продукт интегрируют некоторые функции OpenChat. Компания также представила дорожную карту по развитию OpenCopilot. Пока же разработчик предлагает пользователям встроить флаг функции для OpenCopilot, ограничить его использование несколькими людьми и оценить производительность.OpenAI заявляет, что OpenCopilot распространяется под лицензией MIT, однако пользователи отмечают, что это некая модифицированная лицензия с доступным открытым кодом.В мае Microsoft объявила о запуске Windows Copilot — нового ИИ-помощника, интегрированного прямо в Windows 11. Он будет напоминать ИИ-панели, которые уже доступны в браузере Edge, а также продуктах Office и на платформе GitHub. Обновление Windows 11 23H2 с интегрированным в неё ИИ-помощником Windows Copilot выйдет осенью.
