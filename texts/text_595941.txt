

Вышла публичная бета API для GPT-3 / Habr


              16  December  2021 at 10:47  Вышла публичная бета API для GPT-3 API *Artificial Intelligence Natural Language Processing *      OpenAI объявила о выпуске API своей модели генерации текста GPT-3 в публичную бету. Любой желающий сможет использовать ее для создания приложений.API OpenAI обеспечивает доступ к GPT-3, которая выполняет широкий спектр задач обработки естественного языка, и Codex, которая переводит естественный язык в код, заявила компания.OpenAI предоставит разработчикам инструменты, в том числе бесплатную фильтрацию контента, мониторинг конечных пользователей для предотвращения злоупотреблений, специализированные конечные точки для масштабирования использования API.Теперь разработчики смогут обучать GPT-3 на своих данных, создавая индивидуальную версию, адаптированную к их приложению. Они смогут использовать существующий набор данных практически любой формы и размера или постепенно добавлять данные на основе отзывов пользователей. Для начала им нужно запустить одну команду в инструменте командной строки OpenAI с предоставленным файлом. Как настроить и GPT-3 для приложения:установить клиент на базе openai python из терминала: pip install --upgrade openai;задать ключ API как переменную среды;обучить модель.Компания привела примеры использования GPT-3 в таких проектах, как GitHub Copilot (помогает писать код быстрее с меньшими затратами, используя Codex), Keeper Tax (помогает фрилансерам автоматически находить расходы, не подлежащие вычету из налогооблагаемой базы), Viable (помогает компаниям анализировать отзывы клиентов), Duolingo (исправляет грамматические ошибки при переводе с одного языка на другой).Этим летом Microsoft и GitHub представили нейросетевого помощника программиста Copilot, работающего на базе Codex. Решение OpenAI генерирует код значительно более эффективно, чем GPT-3. Алгоритм обучали на датасете из терабайтов общедоступного кода. Copilot вышел в виде плагина редактора Microsoft Visual Studio Code и функции браузерного редактора кода GitHub Codespaces после регистрации в программе тестирования технического превью сервиса. Также GitHub Copilot уже доступен для работы с JetBrains (в IDE IntelliJ и PyCharm) и в редакторе кода Neovim. После выхода этого сервиса разработчики стали жаловаться, что Copilot вместо нескольких строк нужного кода генерирует десятки строк с цитатами и комментариями из проектов с открытым исходным кодом. Между тем Сбер объявил, что обучил нейросеть ruGPT-3 писать код и назвал разработанную систему JARVIS (Just another really valuable intellectual system).В ноябре OpenAI объявила, что прогресс в обеспечении безопасности позволили компании открыть API GPT-3. Компания отмечала, что разработчики из поддерживаемых стран смогут зарегистрироваться и сразу же начать экспериментировать с ним.
