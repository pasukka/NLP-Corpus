

OpenAI открыла API GPT-3 для всех желающих / Habr


              19  November  2021 at 11:37  OpenAI открыла API GPT-3 для всех желающих API *Artificial Intelligence Natural Language Processing *      OpenAI объявила, что прогресс в обеспечении безопасности позволили компании открыть API генератора текстов GPT-3. Разработчики из поддерживаемых стран смогут зарегистрироваться и сразу же начать экспериментировать с ним.Усовершенствования API GPT-3 за последний год включают модели серии Instruct, специализированные конечные точки для более правдивых ответов на вопросы и бесплатный фильтр контента. Другие изменения включают улучшенную площадку для игр, которая упрощает создание прототипов, библиотеку примеров с десятками подсказок для начала работы разработчиков, и Codex, новую модель, которая переводит естественный язык в код.OpenAI будет проверять приложения с GPT-3 перед их запуском, отслеживать их неправомерное использование и поддерживать разработчиков при масштабировании продуктов.Компания обновила Правила содержания, чтобы уточнить, для какого типа контента может использоваться ее API. Под запрет подпадает разжигающий ненависть, взрослый контент, политика, пропаганда суицида, спам и пр.В OpenAI пообещали, что по мере улучшения безопасности расширят возможности использования GPT-3.В обсуждении отмечается, что это популяризируют новый тип приложений: языковая модель + личная база данных + веб-поиск. Их можно будет использовать для напоминания/обобщения/поиска информации, а также как общий инструмент для исследований.OpenAI показала GPT-3, предназначенный для написания текстов на основе всего нескольких примеров, в конце мая 2020 года. Модель обучали на 175 миллиардов параметрах или 570 гигабайтах текста. GPT-3 может отвечать на вопросы по прочитанному тексту, а также писать стихи, разгадывать анаграммы и осуществлять перевод. Ему достаточно от 10 до 100 примеров для обучения. В отличие от предшественников GPT-2 и GPT-1, исходный код или обучающий набор данных GPT-3 решили не открывать. Модель уже попытались применить в медицинской сфере для общения с пациентами, но результаты эксперимента оказались неутешительными. А в сентябре OpenAI отключил чат-бота на GPT-3, который дал возможность мужчине «поговорить» с покойной невестой.Между тем сама OpenAI представила нейросеть на основе GPT-3, которая способна генерировать краткие изложения книг. Разработчики отмечают, что подобные модели нужны для контролируемого масштабирования систем искусственного интеллекта. Работа данной нейросети покажет, насколько она справляется с работой, которую обычно выполняют люди.
