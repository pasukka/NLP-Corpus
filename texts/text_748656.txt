               Wix анонсировала инструмент на базе ИИ для создания веб-сайтов  Reading time  
    2 min
   Views  1K Website development *E-commerce management *Artificial Intelligence       Компания Wix скоро выпустит инструмент искусственного интеллекта для создания веб-сайтов. AI Site Generator будет следовать инструкциям на естественном языке.Генератор сайтов позволит создавать домашнюю страницу и все внутренние страницы сайта с текстом, изображениями, магазинами, заказами, событиями и многим другим. Результат его работы можно будет отредактировать.Также Wix выпустит инструмент AI Assistant, который будет предлагать улучшения для бизнеса и создаст индивидуальные стратегии на основе аналитики сайта. При этом инструмент AI Page and Section Creator позволит быстро добавить новую страницу или раздел на веб-сайт, а Object Eraser — удалять предметы из изображений.Помимо генератора, Wix разработала для своих клиентов другие инструменты искусственного интеллекта, в том числе:AI Text Creator на базе ChatGPT, который позволяет создавать персонализированный контент для определённых разделов веб-сайта, включая адаптированные заголовки, слоганы и абзацы;AI Template Text Creator для генерации текстов для веб-сайтов. После выбора готового шаблона и ввода краткого описания инструмент генерирует уникальный текстовый контент для всего ресурса, включая главную и внутренние страницы;AI Domain Generator, который помогает пользователям выбрать название своего бренда из различных вариантов запоминающихся и релевантных доменных имен.Кроме того, Wix предлагает инструменты для дизайна на базе ИИ:Alternative Layouts («Альтернативные макеты»), который помогает пользователям изменять макеты на сайте по щелчку мыши;AI Image Creator, который позволяет пользователям сгенерировать изображение по текстовому описанию;Auto Background Removal и Auto Enhance («Автоматическое удаление фона» и «Автоматическое улучшение»), которые улучшают фотографии, чтобы они соответствовали внешнему виду веб-сайта;Auto-Generated Trailer, который превращает видеофайлы в профессиональные короткие ролики.Наконец, Wix разработала инструменты на базе ИИ для нужд электронной коммерции, в том числе генератор описаний продуктов и рекомендательную систему для посетителей сайтов.Ранее компания объявила, что будет продвигать услуги по созданию веб-сайтов, открыв свою серверную часть для разработчиков. В рамках предложения Wix Headless они смогут использовать интерфейсы прикладного программирования (API) и комплекты для разработки программного обеспечения (SDK).
