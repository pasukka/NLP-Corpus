               Hugging Face и ServiceNow выложили в открытый доступ проект StarCoder — аналог GitHub Copilot  Reading time  
    1 min
   Views  4.1K Open source *Programming *GitHub *Artificial Intelligence IT-companies       Стартап Hugging Face и ServiceNow выложили в открытый доступ проект StarCoder — аналог GitHub Copilot. Помощник программиста можно использовать бесплатно.Проект опубликован под лицензией OpenRAIL-M. Настоятельно не рекомендуется использовать ИИ-модель для генерации вредоносного кода.Текущая версия StarCoder с 15 млрд параметров обучена работать на более чем 80 языках программирования, включая данные, тексты и документацию из открытых репозиториев GitHub. Для обучения StarCoder использовался набор данных с открытым исходным кодом под названием The Stack, который содержит более 19 млн тщательно отобранных репозиториев с открытой лицензией и более 6 ТБ кода на более чем 350 языках программирования.Согласно пояснению разработчиков, проект StarCoder совместим с Visual Studio Code. ИИ может отвечать на вопросы о программировании и выполнять базовые запросы пользователей. Например, умеет предоставить необходимые данные на запрос «сгенерируй UI для приложения».Компании Hugging Face и ServiceNow предупредили разработчиков, что StarCoder может выдавать неточные, неполные или сбивающие с толку данные, которые необходимо проверять и дорабатывать. В отличие от GitHub Copilot, у StarCoder меньше функций, а его развитие ещё только началось.
