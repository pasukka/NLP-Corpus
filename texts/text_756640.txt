               Разработчики представили Chaindesk — ноукод-платформу для создания чат-ботов на базе пользовательских данных  Reading time  
    1 min
   Views  573 Machine learning *Artificial Intelligence       Разработчики запустили платформу Chaindesk, позволяющую создавать чат-ботов, обученных на пользовательских данных. Сейчас для обучения можно использовать текст, веб-страницы и файлы из популярных редакторов документов. В будущем появится возможность брать данные из Notion и Airtable.Ноукод-платформа Chaindesk позволяет создавать агентов ChatGPT на базе пользовательских данных. Разработчики предлагают использовать решение в качестве «второго мозга» для работы с документами или для интегрирования чат-ботов в веб-приложения. Для обучения пользователи могут использовать:текст без разметки;веб-страницы;Word;Excel;Powerpoint;PDF;Markdown.Chaindesk можно использовать без навыков программирования. При этом для разработчиков предусмотрено API для интеграции в собственные проекты. Также система автоматически генерирует плагины ChatGPT для каждого отдельного набора данных и указывает ссылку на источник информации.Систему можно запустить локально, но для этого потребуется ключ к OpenAI API. Пользователи без опыта могут воспользоваться уже готовым сервисом. В бесплатном тарифе Discover можно создать одного ИИ-агента с одним набором данных. Также предусмотрены расширенные тарифы Startup, Pro и Enterprise за 25, 99 и 499 долларов в месяц соответственно.Бо
