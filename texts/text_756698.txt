               Dataherald выпустила одноимённый инструмент для поддержки естественного языка в SQL  Reading time  
    1 min
   Views  616 Database Administration *Machine learning *Artificial Intelligence       Dataherald представила ИИ-инструмент для поддержки естественного языка в SQL. Авторы проекта считают, что с его помощью сотрудники без технических навыков смогут оперативно получать данные из базы без помощи аналитиков.Dataherald представляет собой движок естественного языка для SQL на базе искусственного интеллекта, отвечающий на вопросы по структурированным данным. С его помощью можно настроить API на основе корпоративной базы данных, а сотрудники могут задавать вопросы на английском языке. Dataherald можно использовать для:создания решений для сотрудников компании без технических навыков, чтобы они могли быстро получать информацию;интеграции вопросов к базе данных в SaaS-приложении;создания плагинов ChatGPT на основе пользовательских данных.Разработчики инструмента рассказали, что нынешние языковые модели хорошо справляются с написанием SQL-запросов, но не могут должным образом формировать ответы на основе пользовательских данных. Для решения проблемы команда проекта начала разработку Dataherald.Система поддерживает работу с MongoDB, PostgreSQL, BigQuery и Databricks. Также Dataherald модульная, что позволяет тонко настроить систему под нужды конкретного проекта. К примеру, можно подключить визуализацию данных и другие сторонние сервисы.Код Dataherald написан на Python и опубликован на GitHub. Уже сейчас разработчики могут интегрировать систему в свои проекты. На сайте компании доступна документация.
