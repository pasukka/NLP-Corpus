               Microsoft предоставит доступ к технологиям OpenAI всем пользователям своего облачного сервиса Azure  Reading time  
    1 min
   Views  1.7K Cloud services *Artificial Intelligence       Microsoft открыла доступ к сервису Azure OpenAI, который даёт пользователям возможность интегрировать ИИ-инструменты в свои продукты. Компании получат доступ к ИИ-моделям OpenAI, включая GPT-3.5, Codex и DALL-E. Кроме того, скоро в сервисе будет доступен чат-бот ChatGPT, добавил в Twitter гендиректор Microsoft Сатья Наделла.Для доступа к Azure OpenAI необходимо заполнить анкету с указанием рода деятельности компании, описать разрабатываемый продукт и его функции. Кроме того, необходимо в не менее чем пяти предложениях описать сценарий использования ИИ и обозначить его конечных пользователей. Если Microsoft сочтёт ответы неполными или размытыми, а использование ИИ неправомерным, она откажет в доступе.Azure OpenAI год тестировался в ограниченном режиме по нескольким ограниченным сценариям. Microsoft использует его для поддержки GitHub Copilot, ИИ-сервиса на основе OpenAI Codex с функциями автозаполнения фрагментов кода и преобразования комментариев в код. Сервис бизнес-аналитики Microsoft Power BI использует модель GPT-3 для создания формул, а в готовящееся к выпуску приложение Microsoft Designer будет интегрирована DALL-E 2 для создания иллюстраций из текстовых запросов.Microsoft вложила в OpenAI $1 млрд в 2019 году. В 2020 компания приобрела эксклюзивную лицензию на базовую технологию, лежащую в основе GPT-3. С тех пор Microsoft тесно сотрудничает с OpenAI, в числе её недавних планов —  добавить модель преобразования текста в изображение в поисковик Bing. Кроме того, по данным СМИ, Microsoft рассматривает возможность инвестировать в OpenAI $10 млрд. В результате сделки она будет получать 75% прибыли OpenAI, пока инвестиции не окупятся, после чего ей будет принадлежать 49% акций компании.
