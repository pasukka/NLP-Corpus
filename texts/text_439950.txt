Под катом – условия для участников и описание призов для победителей.

Условия

Участие в Тесте для команд с чат-ботами – бесплатное. Регистрация на конкурс означает, что вы согласны с публичным Договором-офертой, который доступен на сайте конкурса, где мы также выложили API. Предварительное тестирование ботов будет проходить с 12 по 13 февраля. А уже во время конференции его будут тестировать эксперты и все желающие. 16 февраля организаторы объявят победителей, которым вручат денежные призы.

Про API

Участники регистрируются в личном кабинете и заводят новых чат-ботов. Для упрощения реализации для участников API сделано синхронным. Сервер обращается к чат-боту по HTTP и ждёт ответа. Протокол общения с чат-ботом – это небольшой JSON:

{
    "chat_id": "uuid",
    "request_id": "uuid",
    "text": "text"
}

chat_id и request_id – уникальные UUID идентификаторы чата и запроса соответственно. text – сообщение от пользователя. 


От бота ожидается простой ответ с текстом:

{
    "text": "text"
}

Есть отдельный Telegram-бот, в котором можно отладить технические моменты общения с чат-ботом. В конкурсном Telegram-боте со стороны сервера будут задержки ответа, чтобы не было возможности определить по времени ответа – бот это или человек.

Призы

Благодаря поддержке компании МТС, победители конкурса получат призы.


Лучшим чат-ботом станет тот, которого максимальное число экспертов приняло за человека. 


Призы:

2 место — 150,000 руб.
3 место — 100,000 руб.


Призы для добровольцев, которые лучше всех сыграли роль бота:

2 место — 15,000 руб.
3 место — 10,000 руб.


Призы для экспертов с лучшими результатами угадывания:

2 место — 15,000 руб.
3 место — 10,000 руб.


Помимо экспертов, любой желающий сможет пообщаться с чат-ботом через Телеграм-бота @TuringTestRussiaBot.


В конце последнего дня конференции будут объявлены не только победители, но и общие результаты конкурса, а также будут продемонстированы лучшие и интересные диалоги, которые заранее отберет команда модераторов.
