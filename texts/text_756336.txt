               Paint в Windows 11 получит опции для рисования с помощью ИИ  Reading time  
    1 min
   Views  1.3K Image processing *Development for Windows *Artificial Intelligence       Microsoft обновит Paint инструментами рисования с помощью ИИ с выходом следующего большого релиза Windows 11. Функции будут основаны на той же технологии, что используется в Bing Image Creator. Инструмент позволяет создавать изображения на основе текстового описания. windowscentral.comВероятно, в Paint появится кнопка «Magic Paint», которую разместят на боковой панели инструментов. Она позволит генерировать картинки с помощью текста. После этого изображение можно будет отредактировать уже на холсте.Также Microsoft планирует добавить инструменты искусственного интеллекта в «Фотографии», приложение «Камера» и «Ножницы». В первом случае ИИ-функция, вероятно, позволит идентифицировать людей или объекты на снимках, а затем вырезать и вставлять их в другие области.В «Ножницах» и «Камере» может появиться технология оптического распознавания символов (OCR). Потенциально это позволит приложениям идентифицировать текст на фотографиях или скриншотах, что значительно упростит копирование и вставку информации из этих файлов.windowscentral.comРанее Microsoft начала тестировать отказ от большего числа встроенных приложений в Windows 11. Так, тестеры теперь могут удалить приложение «Камера», «Фотографии», «Люди» и «Удалённый рабочий стол».
